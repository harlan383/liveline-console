## 异常判定条件

- Shadowrocket 不可用。
- v2rayN 不可用。
- 软路由不可用。
- 普通网页访问异常。
- 目标平台访问异常。
- 直播出现明显卡顿、断流或不稳定。
- `socat` 18443 不稳定。
- `gost` 8443 回退链路不可用。
- 出现完整节点链接、SSH Key、Passphrase、token、服务器密码写入风险。

## 回滚触发条件

- 任一关键客户端不可用。
- 直播出现明显异常。
- 目标平台访问异常。
- `socat` 18443 正式链路不可用。
- 用户人工决定回滚。
- 出现超出 C-minimal 范围的风险。

## 回滚原则

- 如触发回滚，应恢复旧 `node.share_link`。
- `gost` 8443 继续作为回退链路。
- 回滚后必须重新验证客户端可用性。
- 回滚记录不得写入完整节点链接、SSH Key、Passphrase、token、服务器密码或 Reality privateKey。
