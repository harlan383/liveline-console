# Stage 3.3.22 C No-Go Preflight Approval Pack

本阶段只合并 C 方案后续所有“只审批、不执行”的 No-Go 预检事项。本阶段不是正式 cutover，不 SSH，不连接服务器，不执行远程命令，不触发后端任务，不读取或修改正式节点链接。

## 当前阶段结论

- 本阶段是 C No-Go preflight approval pack。
- 本阶段不是正式 cutover。
- 当前仍保持 No-Go。
- 本阶段只合并后续预检审批事项，不执行任何命令。
- 本阶段不 SSH。
- 本阶段不远程。
- 本阶段不读取或修改 `node.share_link`。
- 本阶段不切换端口。
- 本阶段不做 cutover。
- 本阶段不新增数据库迁移。
- 本阶段不新增监听端口。
- 本阶段不关闭、停用或降级 `gost` 8443。
- 本阶段不让 `socat` 接管 `8443`。

## 合并范围说明

本阶段一次性合并以下后续事项：

- 只读预检命令审批。
- 远程只读命令审批。
- 8443 检查审批。
- `socat` 18443 检查审批。
- `gost` 8443 检查审批。
- `node.share_link` 读取审批。
- 后端任务状态查看审批。
- 执行窗口确认。
- 执行人、验收人、回滚负责人确认。
- 命令输出脱敏规则。
- 敏感信息禁止规则。
- 正式 cutover 前最后 No-Go 边界。

本阶段完成后，不再继续拆分“只审批不执行”的小阶段。后续如果继续推进，应进入明确的新阶段：正式 Go 前的最终人工确认，或实际只读预检执行申请。

## 本地只读检查审批

当前仅允许以下本地仓库检查：

- `git status`。
- `git diff`。
- `git log`。
- `git branch`。
- `grep` / `rg` 文档检查。
- docs 阶段文档存在性检查。
- README 阶段状态检查。
- 敏感信息扫描。

这些只限本地仓库，不连接服务器，不触发后端任务，不读取真实数据库，不读取 `.env`、私钥、真实日志或真实凭据。

## 远程只读预检审批

- 当前默认 No-Go。
- 未来如需执行，必须单独审批。
- 必须明确 SSH 账号。
- 必须明确远程目标。
- 必须明确命令白名单。
- 必须明确执行窗口。
- 必须明确输出脱敏规则。
- 本阶段不得执行任何远程只读命令。

## 8443 预检审批

- 当前默认 No-Go。
- 未来如涉及 8443 接管或监听端口变更，必须确认：
  - 云服务器安全组 TCP 8443 是否放行。
  - 云防火墙 TCP 8443 是否放行。
  - 服务器防火墙 TCP 8443 是否放行。
  - 当前监听 8443 的服务。
  - 切换后监听 8443 的服务。
  - `gost` 8443 回滚路径。
- 凡涉及新增、变更监听端口或 8443 接管，必须同步检查云服务器安全组 / 云防火墙 / 服务器防火墙放行对应 TCP 端口。
- 本阶段不执行任何 8443 检查命令。

## `socat` 18443 预检审批

- 当前默认 No-Go。
- 未来如需复核，必须审批以下只读检查：
  - `socat` 18443 是否监听。
  - `socat` 进程是否存在。
  - `socat` 服务状态是否正常。
  - `socat` 转发目标是否符合预期。
  - 18443 候选链路是否仍可用于 Shadowrocket / v2rayN / 软路由。
- 本阶段不执行任何 `socat` 检查命令。

## `gost` 8443 预检审批

- 当前默认 No-Go。
- 未来如需复核，必须审批以下只读检查：
  - `gost` 8443 是否监听。
  - `gost` 进程是否存在。
  - `gost` 服务状态是否正常。
  - `gost` 8443 是否仍作为正式 / 回退链路。
  - `gost` 8443 回滚路径是否存在。
- 本阶段不执行任何 `gost` 检查命令。

## `node.share_link` 审批

- 当前不批准读取真实 `node.share_link`。
- 当前不批准修改 `node.share_link`。
- 未来如需读取，必须单独审批，且输出必须脱敏。
- 未来如需修改，必须确认旧值安全备份、修改目标、回滚方式、客户端回退方式。
- 不得把完整链接写入文档、日志、任务结果或报告。
- 本阶段不读取 `node.share_link`。
- 本阶段不写入 `node.share_link`。
- 本阶段不修改 `node.share_link`。

## 后端任务审批

- 当前不批准查看真实任务状态。
- 当前不批准触发任何后端任务。
- 未来如需查看任务状态，必须单独审批。
- 未来如需触发任务，必须明确任务类型、任务参数、安全边界、回滚方式、任务结果脱敏要求。
- 任务结果不得写入完整节点链接、SSH Key、Passphrase、token、服务器密码。
- 本阶段不查看真实任务状态。
- 本阶段不触发任何后端任务。

## 执行窗口与负责人确认

- 当前不确认验收人。

- 必须明确执行人、验收人、回滚负责人、客户端测试负责人、审批人。

## 输出脱敏规则

- 不得输出真实 SSH Key。
- 不得输出 Passphrase。
- 不得输出 token。
- 不得输出完整节点链接。
- 不得输出服务器密码。
- 不得输出 Reality privateKey。
- IP、UUID、域名、节点链接等如需记录，必须脱敏或使用占位符。
